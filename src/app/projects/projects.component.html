Wifi Weather Station and NeoPixel Controller
Purpose: ECE 476 Advanced Embedded Systems
Hardware and/or Applications: Raspberry Pi Pico and Thonny

The Raspberry Pi Pico broadcasts a wifi signal. Once a device is connect and it navigates to the Pi's IP Address, the Pi transmitts a web page which has temperature, pressure, and light. These are read from sensors connected to the Pico. If the user enters data and hits submit, the Pico changes the color of the Neopixels and can toggle on or off.

GPS Minesweeper Game
ECE 476 Advanced Embedded Systems
Hardware and/or Applications: Raspberry Pi Pico and Thonny

The Raspberry Pi Pico hosts a game that utilizes a GPS signal. The Pico generates a random target. The user drops a pin by pressing a button. The location of this pin is based on GPS signal or joystick input. After a pin is dropped, the Pico draws a circle with the pin in the center and the target someplace on the radius. Other devices can connect to the Pico via bluetooth to track progress. The game is won when a pin is dropped within two meters of the target. Below is a video demostration without GPS. Capture and Compare with Range Senser and Stepper Motor
ECE 376 Embedded Systems
Hardware and/or Applications: PIC18F4620

The PIC measures the distance an object is from the ultrasonic range sensor. Based on this distance, the stepper motor is moved to a certain position.
